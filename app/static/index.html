Simple Restful JSON Api

This is a very lightweight flask application, providing
CRUD endpoints to a json documentstore based on postgresql.
The application will reset it's database upon restart.

Start the application for production by running:
$ gunicorn --bind 0.0.0.0:5000 run:app

For debugging and populating the documentstore with sample data run
$ python3 ./run.py

Available Endpoints

  • GET /documents/id
  • GET /documents
  • POST /documents/create 
  • POST /documents/update/id 
  • GET /documents/delete/id 

Note:

The id is always an integer
The create and update endpoints expect the following format:

{
'data':
{
 'attributes':{
  'doc':{ YOUR : DATA }
 },
 'type':'documents'
}


Example with curl:

$ curl -H "Content-Type: application/json" -X POST -d '{"data":{"attributes":{"doc":{"jobname":"foo","status":"open"}}, "type":"documents"}}' http://example.com/documents/create
